# 改善内容連絡書 管理システム仕様書

## 1. 概要

本システムは、改善内容の連絡書に対し、編集、登録、承認、展開、実施、結果回収、クローズまでの一連の流れをVBAで管理するシステムである。

従来の紙・電子・コピーが混在した運用を統一し、回覧状況および進捗を台帳で確認できることを目的とする。

## 2. 目的

-   改善内容連絡書の運用方法を統一する
-   承認・配布・実施の進捗を見える化する
-   作成からクローズまでを一元管理する

## 3. 全体フロー

編集 → 登録 → 配信設定 → 発行部署承認 → 品証承認 → 担当戻り →
実施部署展開 → 実施部署承認 → 実施 → 結果回収 → クローズ承認 → クローズ

## 4. 機能一覧

1.  編集機能
2.  登録機能
3.  配信設定機能
4.  承認・配信機能
5.  発行部署承認機能
6.  品証承認機能
7.  実施部署展開機能
8.  実施部署承認機能
9.  実施管理機能
10. 結果回収機能
11. クローズ承認機能
12. クローズ機能

## 5. 承認ルート

発行部署：担当 → 総括 → 課長\
品証部署：担当/工長 → 係長 → 課長 → 担当\
実施部署：課長 → 係長 → 工長 → 実施\
実施後：工長 → 係長 → 課長 → 担当\
クローズ：担当 → 総括 → 課長 → 担当 → クローズ

## 6. 処理仕様

承認と配信は同一ボタンで実行され、承認と同時に次承認者へメール送信される。

## 7. 台帳仕様

管理番号 / 件名 / 作成日 / 作成者 / 対象部署 / ステータス / 各承認日 /
クローズ日

## 8. 効果

-   運用統一
-   見える化
-   作業削減
-   一連管理

## 9. 未対応

-   例外処理
-   差戻し
-   複数部署制御
