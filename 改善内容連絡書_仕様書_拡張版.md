# 改善内容連絡書 管理システム仕様書（拡張版）

## 1. 概要

本システムは、改善内容の連絡書に対し、編集、登録、承認、展開、実施、結果回収、クローズまでの一連の流れをVBAで管理するシステムである。

## 2. 目的

-   運用方法の統一
-   状況の見える化
-   一元管理

## 3. 全体フロー

編集 → 登録 → 配信設定 → 承認 → 展開 → 実施 → 結果回収 → クローズ

## 4. 状態遷移図

``` mermaid
stateDiagram-v2
[*] --> 編集中
編集中 --> 登録済 : 登録
登録済 --> 発行部署承認中 : 配信開始
発行部署承認中 --> 品証承認中
品証承認中 --> 担当戻り
担当戻り --> 実施部署展開中
実施部署展開中 --> 実施部署承認中
実施部署承認中 --> 実施中
実施中 --> 結果承認中
結果承認中 --> クローズ承認中
クローズ承認中 --> クローズ
クローズ --> [*]
```

## 5. 状態一覧

  状態             内容
  ---------------- --------------
  編集中           作成中
  登録済           登録完了
  発行部署承認中   承認中
  品証承認中       品証承認中
  担当戻り         担当へ戻る
  実施部署展開中   展開処理中
  実施部署承認中   実施部署承認
  実施中           実施作業
  結果承認中       実施後承認
  クローズ承認中   最終承認
  クローズ         完了

## 6. 例外処理

-   承認者不在：代理またはスキップ
-   差戻し：前工程へ戻す
-   複数部署：全完了で次工程
-   未確認：再通知
-   修正：承認リセット
-   再開：クローズ解除

## 7. 効果

-   運用統一
-   見える化
-   作業削減
